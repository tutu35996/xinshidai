# Templates

## File Structure

```
templates/
├── english/
├── arabic/
├── armenian/
├── croatian/
├── dutch/
├── filipino/
├── french/
├── russian/
├── spanish/
└── vietnamese/
```

## File Naming

Each folder contains VIP1-VIP10 template files:
- `{Language}_vip1.png`
- `{Language}_vip2.png`
- ...
- `{Language}_vip10.png`

## Requirements

- Format: PNG
- Size: 1280 x 800 pixels
